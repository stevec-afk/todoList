# Todo List

Live Demo: [[https://github.io](https://stevec-afk.github.io/todoList/)]

## Overview
A task management application built as part of The Odin Project curriculum. The app focuses on a clean, minimalist UI with a fluid layout that adapts between mobile and desktop views.

## Key Features
- Task Organization: Create, edit, and delete tasks with specific due dates and categories.
- Fluid Responsive Design: Custom-built CSS using rem/clamp units for seamless scaling across devices.
- Dynamic Filtering: View tasks by date (Today, Tomorrow, Month) or user-defined categories.
- Persistence: All tasks and user preferences (Theme, Visibility) are saved to LocalStorage.
- Theme Support: Full light and dark mode toggles.

## Tech Stack
- Vanilla JavaScript (ES6 Modules)
- HTML5 / CSS3 (Custom Variables and Grid/Flexbox)
- Webpack (Asset bundling and deployment)
- date-fns (Date manipulation)

## Development
To run this project locally:
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the development server.
4. Run `npm run build` to generate the production bundle in the dist folder.
