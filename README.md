# TodoList
Todo list

# Planning notes
Features: 
- Create todo items
- Categories
- Calendar / list by date view 
- localstorage
- Expand / collapse details
- Edit & delete buttons

Todo properties: 
- Title
- Target completion dates (optional)
- Description
- Completed? true/false toggle
- Project / category

Useful NPM modules: 
- date-fns 

# Research notes
- side panel should be fixed, no need to pop in an out
- 1 todo = 1 line
- All tasks means all tasks, including completed
- Button to reset data 
- Material icons
- Built by & source links bottom left
- No "inbox" - its not email.
- No fake "user" - there's no login, only localstorage

# Extra features 
- Light / dark mode 
- Mobile friendly
